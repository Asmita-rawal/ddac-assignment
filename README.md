## ✨ Features

- ✅ **3-Step Registration Form**
  - Step 1: Personal Information (Name, Email, Phone, Role)
  - Step 2: Account Setup (Password, Security Question)
  - Step 3: Confirm & Submit
- ✅ **Password Strength Indicator**
- ✅ **Password Confirmation Validation**
- ✅ **Email Validation**
- ✅ **Secure Password Hashing** (bcrypt)
- ✅ **MySQL Database Storage**
- ✅ **RESTful API**
- ✅ **Toast Notifications**
- ✅ **Responsive Design** (Mobile-friendly)
- ✅ **User Data Viewing API**

## 🛠️ Technologies Used

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **MySQL2** - MySQL driver
- **bcryptjs** - Password hashing
- **CORS** - Cross-origin resource sharing
- **Nodemon** - Development auto-reload

### Frontend
- **HTML5** - Structure
- **CSS3** - Styling with animations
- **Vanilla JavaScript** - Form logic and API calls
- **Font Awesome** - Icons
- **Google Fonts** - Typography

### Database
- **MySQL** - Relational database
- **SQL** - Data storage and queries
